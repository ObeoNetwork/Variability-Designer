How to use Product Derivation

To launch Product Derivation from the Feature Diagram Editor simply click on the FeatureDiagram (white part ) ->Derive Feature Model -> Derive Feature Model.

As shown on the following picture, you can choose the feature you want on the Eclipse console :

Product Derivation is made in two steps :

  1. Call selection engine asking user to select feature :

    To do this , simply right click on the white part of the FeatureDiagram -> Selection Engine -> Selection. Then :

    1. Select file name for resolution model.

      resolution model must have .resolutionmodel extension

      Do not forget to use the extension .resolutionmodel otherwise, product derivation cannot be made correctly.

    2. The selection engine browses the model starting with all children of a given feature and then the children of the selected features. On this step you can determines how many choices you want by typing the number of desired choices and select a proposed feature by typing its name on the Eclipse Console. •

  2. Then you need to launch derivation.As for selection engine, right click on the white part of the FeatureDiagram -> Product Derivation -> Derive Feature Model and type to it the path for the wanted resolution model.

  3. Type the name of the product model (output model for product derivation process).

all models used in selection and product derivation are stored on the same folder as feature model

Note that in this version all models used in selection and product derivation are stored on the same folder as feature model

So, you have obtained a new product model by this way.

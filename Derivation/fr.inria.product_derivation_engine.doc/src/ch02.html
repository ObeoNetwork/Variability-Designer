Chapter 2. Installation

You need to install first the Feature Diagram Editor and all its dependencies. For more informations about that please refer to the Feature Diagram Editor Documentation.

To install the Product Derivation Engine unzip the update site fr.inriaproduct_derivation_engine.updatesite, open Eclipse and go to Help -> Install New Software. On the new wizard, click on Add name your new update site Product Derivation Engine for Feature Diagram Editor and search the unzipped update site.

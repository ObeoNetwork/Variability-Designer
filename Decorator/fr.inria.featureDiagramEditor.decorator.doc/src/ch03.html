Enable use of product derivation engine

We suppose you have ever created a feature model and associate Domain Model Elements on it as presented on the following figure :

You need to add the viewpoint "Design for Product Derivation" on the session file fd.aird. To do this, go to the Model Contents view, click on Local session : fd.aird and on Viewpoints Selection. You obtain the following wizard :

You need to check both Design viewpoint (viewpoint for Feature Diagram Editor) and Design for Product Derivation viewpoint ( viewpoint that add a popup menu on Feature Diagram Editor to launch Product Derivation).

However, to activate Design for Product Derivation layer click on the icon outlined in red on the figure above and check Product Derivation.
